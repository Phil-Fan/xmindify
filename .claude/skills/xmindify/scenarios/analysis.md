# 问题分析与决策

## 适用场景

问题根因分析、方案对比决策、利弊权衡

## 问题分析模板

```
问题描述

- 问题描述
    * 现象
    * 影响范围
    * 紧急程度

- 原因分析 [1][B]
    * 直接原因
        - 表现
        - 触发条件
    * 根本原因
        - 深层因素
        - 系统性问题
    * 外部因素
        - 环境变化
        - 第三方影响

- 影响分析 [^1](leads to)[B]
    * 用户影响
    * 业务影响
    * 技术债务

- 解决方案 [2][B]
    * 方案A
        - 优点
        - 缺点
        - 成本
    * 方案B
        - 优点
        - 缺点
        - 成本
    * 方案C [^2](recommended)

- 决策 [S]
    * 选择方案
    * 理由
    * 风险预案
    [S]: 最终决策
```

## 方案对比模板

```
决策主题

- 方案A [B1]
    * 功能特性
    * 成本预算
    * 实施难度
    * 预期收益

- 方案B [B2]
    * 功能特性
    * 成本预算
    * 实施难度
    * 预期收益

- 对比维度 [B]
    * 维度1 [1]
        - 方案A表现
        - 方案B表现
    * 维度2
        - 方案A表现
        - 方案B表现
    * 维度3 [^1](decisive)
        - 方案A表现
        - 方案B表现

- 推荐 [S]
    * 推荐方案
    * 核心理由
    * 风险提示
    [S]: 推荐结论
```
