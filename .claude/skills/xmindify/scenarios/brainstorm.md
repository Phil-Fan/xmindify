# 头脑风暴

## 适用场景

创意构思、方案设计、发散思维

## 通用模板

```
中心主题

- 维度1 [B]
    * 创意A
    * 创意B
    * 创意C

- 维度2 [B]
    * 创意D [1]
    * 创意E
    * 创意F

- 维度3 [B]
    * 创意G [^1](related)
    * 创意H
    * 创意I

- 优先方案 [S]
    * 方案1
    * 方案2
    * 方案3
    [S]: 推荐方案

- 下一步行动
    * 立即执行
    * 需要验证
    * 后续跟进
```

## 5W2H 分析法

```
问题/目标

- What (是什么)
    * 核心问题
    * 涉及对象
    * 关键要素

- Why (为什么)
    * 根本原因
    * 动机/目标
    * 价值意义

- Who (谁)
    * 相关人员
    * 目标用户
    * 利益相关者

- When (何时)
    * 时间节点
    * 里程碑
    * 紧急程度

- Where (何地)
    * 发生场所
    * 影响范围
    * 资源位置

- How (如何)
    * 实施方法 [1]
    * 技术路径
    * 操作流程 [^1](alternative)

- How much (多少)
    * 成本预算
    * 资源需求
    * 预期收益
```

## SWOT 分析

```
分析主题

- 优势 (Strengths) [B1]
    * 资源优势
    * 能力优势
    * 竞争优势

- 劣势 (Weaknesses) [B1]
    * 资源短板
    * 能力不足
    * 竞争劣势

- 机会 (Opportunities) [B2]
    * 市场机会
    * 政策支持
    * 技术趋势 [1]

- 威胁 (Threats) [B2]
    * 竞争压力
    * 市场风险
    * 技术挑战 [^1](mitigated by)

- 战略建议 [S]
    * SO战略 (优势+机会)
    * WO战略 (机会+劣势)
    * ST战略 (优势+威胁)
    * WT战略 (劣势+威胁)
    [S]: 综合策略
```
