# 内容总结提炼

## 适用场景

文章总结、报告提炼、书籍要点梳理

## XMindMark 模板

```
内容标题

- 概览 [B]
    * 内容类型
    * 核心主题
    * 目标读者
    * 关键数据

- 核心论点 [1][B]
    * 主要观点1
    * 主要观点2
    * 主要观点3

- 支撑论据 [^1](supported by)[B]
    * 事实/数据
    * 案例分析
    * 专家观点
    * 研究结果

- 关键结论 [S1]
    * 结论1
    * 结论2
    * 结论3
    [S1]: 核心结论

- 行动建议 [S2]
    * 立即可做
    * 短期规划
    * 长期策略
    [S2]: 行动清单

- 延伸思考
    * 相关议题
    * 开放问题
    * 个人见解

- 资源链接
    * 原文链接
    * 参考资料
    * 相关阅读
```

## 结构建议

- 每个分支 3-7 个要点
- 层级不超过 3 层
- 使用边界 `[B]` 逻辑分组
- 使用总结 `[S]` 汇总关键要点
