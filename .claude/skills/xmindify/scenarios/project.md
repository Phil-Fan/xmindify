# 项目规划

## 适用场景

项目计划、任务分解、路线图设计

## 项目规划模板

```
项目名称

- 项目概览 [B]
    * 项目目标
    * 成功标准
    * 时间范围
    * 资源约束

- 阶段规划 [B]
    * 阶段一：启动 [1]
        - 里程碑1
        - 交付物
    * 阶段二：执行 [^1](depends on)
        - 里程碑2
        - 交付物
    * 阶段三：收尾
        - 里程碑3
        - 交付物

- 关键任务 [B]
    * 任务类别1
        - 任务1.1
        - 任务1.2
    * 任务类别2
        - 任务2.1
        - 任务2.2

- 资源分配 [B]
    * 人力资源
    * 技术资源
    * 预算分配

- 风险管理 [B]
    * 风险识别
    * 应对措施
    * 备选方案

- 里程碑检查点 [S]
    * 检查点1
    * 检查点2
    * 检查点3
    [S]: 关键里程碑
```

## OKR 规划模板

```
周期：YYYY-QX

- Objective 1
    * KR1 [1]
        - 当前状态
        - 目标值
    * KR2
        - 当前状态
        - 目标值
    * Key Actions [^1](contributes to)
        - 行动1
        - 行动2

- Objective 2
    * KR1 [2]
        - 当前状态
        - 目标值
    * KR2
        - 当前状态
        - 目标值
    * Key Actions [^2](supports)
        - 行动1
        - 行动2

- 资源需求 [B]
    * 人力
    * 预算
    * 技术支持

- 潜在风险 [B]
    * 风险1
    * 风险2
```
